# Commit Standard (from v11)

This repository standardizes commit format as follows:

add vX.X.X code (core: ...)
add vX.X.X memo (state: ...)
add vX.X.X result (status: ...)

- code: what was implemented
- memo: meaning / phase result
- result: observed behavior

This standard applies from v11 onward.
v10 and earlier commits are intentionally left as-is to preserve experimental history.
